### 🔭 Currently building

**NickoScope32** — my flagship: a modern **CRT vector display platform**. It steers the
X/Y deflection plates of an oscilloscope CRT tube to draw graphics and clock faces through
**analog waveform synthesis** — a real glowing vector display, no pixels. *Open hardware, in active development.*
