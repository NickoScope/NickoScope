### 🔭 Currently building

**[nickol-knx-mcp](https://github.com/NickoScope/nickol-knx-mcp)** — a design-time **KNX / ETS6
assistant as an MCP server** (25 tools): design a project from a spec into a full implementation
kit, audit & repair existing `.knxproj` files, generate the Home Assistant layer — with exact
vendor device models parsed straight from ETS application programs. *Read-only by design: it can
never touch a live bus.*

**NickoScope32** — a modern **CRT vector display platform**. It steers the X/Y deflection plates
of an oscilloscope CRT tube to draw graphics and clock faces through **analog waveform
synthesis** — a real glowing vector display, no pixels. *Open hardware, in active development.*
